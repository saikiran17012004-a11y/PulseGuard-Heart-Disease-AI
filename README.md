# ❤️ PulseGuard AI: Heart Disease Diagnostic System

PulseGuard AI is an end-to-end Machine Learning project that predicts the probability of heart disease using clinical patient data. This project highlights my transition from **Mechanical Engineering** to **AI & Data Science**, demonstrating skills in data preprocessing, predictive modeling, and web deployment.

## 🚀 [Live Demo Link](INSERT_YOUR_STREAMLIT_URL_HERE)

## 🛠️ Key Features
- **Predictive Analytics:** Uses a Logistic Regression model to identify heart disease risk.
- **Medical Dashboard:** Interactive UI built with Streamlit for real-time diagnostics.
- **Robust Preprocessing:** Handles missing values and utilizes **StandardScaler** and **Label Encoding** for high-accuracy predictions.
- **Normalization:** Custom logic to handle case-sensitive text inputs to ensure application stability.

## 📊 Dataset: UCI Heart Disease
The model is trained on 900+ patient records using features such as:
- Age & Sex
- Chest Pain Type (Typical, Asymptomatic, etc.)
- Resting Blood Pressure & Cholesterol
- Max Heart Rate (Thalch)
- ST Depression (Oldpeak)

## 💻 Tech Stack
- **Language:** Python 3.13
- **Libraries:** Scikit-Learn, Pandas, NumPy, Joblib
- **Deployment:** Streamlit Cloud
- **Version Control:** Git & GitHub Desktop

## 📂 File Structure
- `app.py`: The live web application code.
- `heart_train.py`: Data cleaning and model training script.
- `heart_disease_uci.csv.csv`: The raw dataset used for training.
- `*.pkl`: Serialized model, scaler, and encoder files (the "AI Brain").
